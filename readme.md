# Helin Tunalı – Performance Marketing Antigravity Workspace

Bu repo, Helin Tunalı'nın Performance Marketing → Growth Specialist yolculuğunu hızlandırmak için oluşturulmuş bir **Google Antigravity Workspace**’idir.

Antigravity içindeki agent’lar, bu repo üzerinden:

- Haftalık öğrenme görevleri
- Kampanya simülasyonları
- Meta / Google test planları
- KPI analizleri
- Kariyer gelişimi önerileri
- Yol haritası bazlı görevler
- Aylık performans review çıktıları
- Creative brief + Funnel önerileri

üretebilir.
